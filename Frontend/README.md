🎨 Frontend - Image Posting App

This is the frontend of the Image Posting App built using React and Tailwind CSS.

🚀 Features

- 🔐 Login / Signup UI
- 🖼️ Image feed
- ❤️ Like animation (double click)
- ⚡ GSAP animations (stairs effect)
- 📱 Fully responsive design

🛠️ Tech Stack

- React.js
- Tailwind CSS
- Axios
- GSAP

📂 Folder Structure

frontend/
│── src/
│   │── components/
│   │── pages/
│   │── hooks/
│   │── App.jsx
│   │── main.jsx

⚙️ Setup

Install dependencies

npm install

Run frontend

npm run dev

🔗 API Connection

Make sure backend is running and update base URL:

http://localhost:3000/api

🎬 Animations

- GSAP used for:
  - Stair animation
  - Like button animation

📸 Features UI

- Image cards
- Like button ❤️
- Upload form
- Authentication pages

📌 Notes

- Ensure backend is running before frontend
- Use ".env" for API URL if needed
